# WorkIndia DSM General Documentation

This directory contains general documentation for WorkIndia Design System.

## Available Topics

### Usage

The Usage guide covers the core fundamentals of implementing WorkIndia DSM in your application. It includes setup instructions for `WorkIndiaProvider` and how to do initial setup of WorkIndia DSM in your app.

### AvailableIcons

A reference of which all icons are available in the WorkIndia DSM. This includes usage examples showing how to import icons, use them in components with icon props, and implement them as standalone components.
