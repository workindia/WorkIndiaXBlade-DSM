## Component Name

MagnifyingGlassPlusFilIcon

## Description

Zoom in, enlarge, or search more specifically. Use for zoom in buttons, search with more detail, or expanding view. Filled variants provide more emphasis.

## Category

Plus

## Variant

Standard

## When to Use

- Add or create actions
- Increase quantity
- Positive or yes responses

## When Not to Use

- For remove (use MinusIcon)
- For expand (use CaretDownIcon)

## Import

```typescript
import { MagnifyingGlassPlusFilIcon } from '@workindia/dsm';
```

## Usage

```tsx
import { MagnifyingGlassPlusFilIcon } from '@workindia/dsm';

const Example = () => {
  return (
    <MagnifyingGlassPlusFilIcon
      size="medium"
      color="surface.icon.primary.normal"
    />
  );
};
```

## Design Guidelines

- Use icons consistently throughout your application
- Maintain appropriate size hierarchy (small: 16px, medium: 20px, large: 24px, xlarge: 32px)
- Use semantic colors from the design tokens
- Ensure sufficient contrast for accessibility
- Use filled variants for primary actions and outline variants for secondary actions
