## Component Name

TrendDownIcon

## Description

Represents trends, growth, decline, or analytics.

Standard icons provide a balanced appearance suitable for most use cases.

## Category

Trend

## Variant

Standard

## When to Use

- Trend indicators
- Analytics or statistics
- Growth or decline features

## When Not to Use

- For arrows in general
- For navigation

## Import

```typescript
import { TrendDownIcon } from '@workindia/dsm';
```

## Usage

```tsx
import { TrendDownIcon } from '@workindia/dsm';

const Example = () => {
  return <TrendDownIcon size="medium" color="surface.icon.primary.normal" />;
};
```

## Related Icons

TrendDownFillIcon, TrendDownOutlineIcon, TrendDownIcon

## Design Guidelines

- Use icons consistently throughout your application
- Maintain appropriate size hierarchy (small: 16px, medium: 20px, large: 24px, xlarge: 32px)
- Use semantic colors from the design tokens
- Ensure sufficient contrast for accessibility
- Use filled variants for primary actions and outline variants for secondary actions
