## Component Name

HandArrowUpIcon

## Description

Displays a hand pointing upward, representing offer, promote, or hire action.

Standard icons provide a balanced appearance suitable for most use cases.

## When to Use

- Offer letters
- Promotion actions
- Hire decisions
- Elevate content

## When Not to Use

- As decorative elements
- For indicating links (use LinkIcon)

## Import

```typescript
import { HandArrowUpIcon } from '@workindia/dsm';
```

## Usage

```tsx
import { HandArrowUpIcon } from '@workindia/dsm';

const Example = () => {
  return <HandArrowUpIcon size="medium" color="surface.icon.primary.normal" />;
};
```

## Design Guidelines

- Use icons consistently throughout your application
- Maintain appropriate size hierarchy (small: 16px, medium: 20px, large: 24px, xlarge: 32px)
- Use semantic colors from the design tokens
- Ensure sufficient contrast for accessibility
- Use filled variants for primary actions and outline variants for secondary actions
