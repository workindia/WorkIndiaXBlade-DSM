## Component Name

FlowerTulipIcon

## Description

Displays a tulip flower, representing nature, plants, or spring theme.

Standard icons provide a balanced appearance suitable for most use cases.

## When to Use

- Nature themes
- Plant-related features
- Garden features
- Spring campaigns

## When Not to Use

- When the icon meaning does not match your feature
- As a decorative element without proper context

## Import

```typescript
import { FlowerTulipIcon } from '@workindia/dsm';
```

## Usage

```tsx
import { FlowerTulipIcon } from '@workindia/dsm';

const Example = () => {
  return <FlowerTulipIcon size="medium" color="surface.icon.primary.normal" />;
};
```

## Design Guidelines

- Use icons consistently throughout your application
- Maintain appropriate size hierarchy (small: 16px, medium: 20px, large: 24px, xlarge: 32px)
- Use semantic colors from the design tokens
- Ensure sufficient contrast for accessibility
- Use filled variants for primary actions and outline variants for secondary actions
