## Component Name

InfinityIcon

## Description

Represents endless, infinite, or unlimited options.

Standard icons provide a balanced appearance suitable for most use cases.

## Category

Infinity

## Variant

Standard

## When to Use

- Unlimited features
- Infinite scrolling
- Looping or continuous content

## When Not to Use

- As a decorative element
- For mathematical symbols

## Import

```typescript
import { InfinityIcon } from '@workindia/dsm';
```

## Usage

```tsx
import { InfinityIcon } from '@workindia/dsm';

const Example = () => {
  return <InfinityIcon size="medium" color="surface.icon.primary.normal" />;
};
```

## Related Icons

InfinityFillIcon, InfinityOutlineIcon, InfinityIcon

## Design Guidelines

- Use icons consistently throughout your application
- Maintain appropriate size hierarchy (small: 16px, medium: 20px, large: 24px, xlarge: 32px)
- Use semantic colors from the design tokens
- Ensure sufficient contrast for accessibility
- Use filled variants for primary actions and outline variants for secondary actions
